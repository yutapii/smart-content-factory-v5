# 🔑 Google Cloud Vision API 認証ファイル取得ガイド

## 📝 ステップバイステップ手順

### 1️⃣ Google Cloud Consoleにアクセス
👉 https://console.cloud.google.com/

### 2️⃣ プロジェクトを作成（または既存のプロジェクトを選択）
- 上部のプロジェクトセレクターをクリック
- 「新しいプロジェクト」をクリック
- プロジェクト名を入力（例：`smart-content-factory`）
- 「作成」をクリック

### 3️⃣ Vision APIを有効化
- 左メニューから「APIとサービス」→「ライブラリ」
- 検索バーに「Vision API」と入力
- 「Cloud Vision API」をクリック
- 「有効にする」ボタンをクリック

### 4️⃣ サービスアカウントを作成
- 左メニューから「IAMと管理」→「サービスアカウント」
- 「サービスアカウントを作成」をクリック
- サービスアカウント名：`vision-api-service`
- 「作成して続行」をクリック

### 5️⃣ ロール（権限）を設定
- ロールで「Cloud Vision」→「Cloud Vision API ユーザー」を選択
- 「続行」をクリック
- 「完了」をクリック

### 6️⃣ キーを作成してダウンロード
- 作成したサービスアカウントをクリック
- 「キー」タブを選択
- 「鍵を追加」→「新しい鍵を作成」
- 「JSON」を選択
- 「作成」をクリック
- **JSONファイルが自動的にダウンロードされます** 📥

### 7️⃣ ダウンロードしたJSONファイルを保存
推奨保存場所：
```bash
# ホームディレクトリに credentials フォルダを作成
mkdir -p ~/credentials
mv ~/Downloads/*.json ~/credentials/google-vision-api.json
```

## 🚀 OCRサーバーを起動

認証ファイルをダウンロードしたら：

```bash
cd /Users/saitoyutaka/smart-content-factory-v5
./start_ocr_with_credentials.sh
# プロンプトでダウンロードしたJSONファイルのパスを指定
```

## ⚠️ 重要な注意事項

1. **JSONファイルは絶対に公開しない**
   - GitHubにアップロードしない
   - SNSで共有しない
   
2. **安全な場所に保管**
   - `~/credentials/` フォルダなど
   - バックアップも取っておく

3. **不要になったら削除**
   - プロジェクトが終わったらキーを無効化

## 🆘 困ったときは

### Q: プロジェクトが作れない
A: 無料のGoogleアカウントでOK。クレジットカード登録は不要（無料枠内なら）

### Q: APIが有効にできない
A: 課金設定が必要な場合があります。でも実際に料金が発生するのは無料枠を超えてから

### Q: JSONファイルをなくした
A: サービスアカウントから新しいキーを作成すればOK
